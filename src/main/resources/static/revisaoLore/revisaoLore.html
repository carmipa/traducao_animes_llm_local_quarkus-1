Revisão de Lore

Padroniza nomes de personagens, locais, facções, mechas e termos de mundo conforme a lore oficial da obra. Escolha a aba conforme você tenha (ou não) as legendas originais em inglês.

Selecione obrigatoriamente a obra no menu abaixo — isso carrega a lore canônica e exibe os metadados do anime/filme via API.

Compara EN × PT-BR via LLM local: máxima precisão, inclusive nos homógrafos.

Sem o inglês: a camada determinística corrige só os termos inequívocos (multi-palavra, ex.: "Traje Móvel" → "Mobile Suit"); ative o LLM para tentar também os homógrafos de uma palavra (ex.: "Eixo"/"Axis"). Toda alteração faz backup automático.
